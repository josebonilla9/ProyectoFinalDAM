Instrucciones y requisitos para hacer funcionar la aplicación

Requisitos:
    - Usar IDE NetBeans
    - Instalar XAMPP
    - Descargar todos los archivos
    - Añadir jCalendar al proyecto si no está instalado (archivo .jar incluido)

Instrucciones:
    - Instalar base de datos:
        1º Cambiar puertos de la base de datos MySQL de XAMPP a 3308
        2º Cambiar puerto de Apache Web Server a 50
        3º Importar base de datos en phpMyAdmin desde el archivo tfgdam.sql

    - Instalar jCalendar:
        1º Ir a GraphicInterface.java
        2º Botón derecho en Palette y pulsar en Palette Manager
        3º Pulsar en Add from JAR y buscar el archivo jCalendar.jar
        4º Elegir todos los componentes y añadirlos a una carpeta
